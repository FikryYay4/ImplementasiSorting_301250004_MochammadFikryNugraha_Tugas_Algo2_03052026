## 📌 Deskripsi

Project ini merupakan implementasi beberapa algoritma **sorting (pengurutan data)** dalam ilmu komputer.

Sorting digunakan untuk mengurutkan data agar lebih mudah diproses, dicari, dan dianalisis.

Program dibuat menggunakan **Python** dengan konsep **modular programming**, sehingga kode lebih terstruktur, rapi, dan scalable.

---

## 🧠 Konsep Algoritma

### 🔁 Bubble Sort
- Membandingkan elemen bersebelahan
- Menukar posisi jika urutan salah
- Diulang sampai data terurut

---

### 🔀 Selection Sort
- Mencari nilai minimum
- Menempatkan ke posisi awal
- Diulang untuk seluruh data

---

### 🧩 Insertion Sort
- Menyisipkan elemen ke posisi yang tepat
- Mirip seperti menyusun kartu
- Efisien untuk data kecil

## 🔄 IPO (Input - Process - Output)

### 🟢 Input
- Jumlah data
- Nilai data (input menggunakan looping)

### 🔵 Process
- Bubble Sort → pertukaran berulang
- Selection Sort → pemilihan nilai minimum
- Insertion Sort → penyisipan elemen

### 🔴 Output
- Data yang sudah terurut (ascending / descending)
